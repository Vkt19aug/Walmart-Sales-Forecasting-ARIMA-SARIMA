# 📊 Walmart Sales Forecasting (Time Series Analysis)

## 📌 Overview
This project focuses on forecasting Walmart weekly sales using time series analysis techniques. It includes end-to-end data processing, exploratory data analysis (EDA), and implementation of forecasting models to identify trends and seasonality.

## 🎯 Objective
- Analyze Walmart sales data
- Identify trends and seasonal patterns
- Build forecasting models using time series techniques
- Generate insights for better business decision-making

## 🛠️ Tech Stack
- Python (Pandas, NumPy)
- Matplotlib & Seaborn
- Statsmodels (AR, ARIMA, SARIMA)
- Jupyter Notebook

## 🔍 Key Analysis
- Data Cleaning and Preprocessing
- Time Series Decomposition
- Stationarity Check (ADF Test)
- Trend and Seasonality Analysis
- Model Building (AR, ARIMA, SARIMA)
- Forecasting Future Sales

## 📊 Models Used
- Auto Regression (AR)
- ARIMA (AutoRegressive Integrated Moving Average)
- SARIMA (Seasonal ARIMA)

## 📈 Key Insights
- Identified strong seasonal patterns in sales
- Observed trends in weekly sales over time
- SARIMA model performed better for capturing seasonality
- Forecasting helps in inventory and demand planning
